### Built With

<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites
  ```sh
   npm install -g expo-cli
   ```
   
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/abhey-sujith/note-app-react-native
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
